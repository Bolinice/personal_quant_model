# 前台 UI/UX 设计说明书

> 版本：V1.0 | 日期：2026-04-26 | 状态：规划中

---

## 1. 设计原则

### 1.1 核心理念

- **专业可信**：面向专业投资者，界面风格简洁、数据密度高
- **合规优先**：关键位置嵌入免责声明，避免误导性展示
- **渐进式复杂度**：新手可快速上手，专业用户可深度使用
- **响应式设计**：支持桌面端（主）与移动端（辅）

### 1.2 设计规范

| 维度 | 规范 |
|------|------|
| 主色调 | 深蓝 #1A365D（专业感） |
| 辅助色 | 绿 #38A169（涨）/ 红 #E53E3E（跌） |
| 字体 | 系统默认 + 数字等宽字体 (Tabular Nums) |
| 间距 | 8px 基础网格 |
| 圆角 | 4px（紧凑专业） |
| 阴影 | 轻阴影，避免过度装饰 |

## 4. 组件规范

### 4.1 数据展示组件

| 组件 | 用途 | 规范 |
|------|------|------|
| `DataCard` | 关键指标展示 | 标题 + 数值 + 变化率 + 迷你图 |
| `FactorTable` | 因子列表 | 可排序、可筛选、支持分页 |
| `ReturnChart` | 收益曲线 | 策略 vs 基准，含回撤区域 |
| `HeatMap` | IC 热力图 | 横轴时间、纵轴因子 |
| `RiskMeter` | 风险等级 | 5 级刻度，颜色渐变 |

### 4.2 交互组件

| 组件 | 用途 | 规范 |
|------|------|------|
| `BacktestForm` | 回测参数配置 | 步骤式表单，参数校验 |
| `FactorFilter` | 因子筛选 | 多维筛选 + 搜索 |
| `DateRangePicker` | 日期范围 | 快捷选项 + 自定义 |
| `ExportButton` | 数据导出 | CSV/Excel，含免责水印 |

### 4.3 合规组件

| 组件 | 用途 | 展示规则 |
|------|------|----------|
| `DisclaimerBanner` | 免责横幅 | 所有含收益数据页面底部 |
| `RiskBadge` | 风险标签 | 策略/组合卡片上显示风险等级 |
| `BacktestTag` | 回测标记 | 所有历史数据旁标注"回测" |
| `ComplianceModal` | 合规弹窗 | 首次访问/关键操作前弹出 |

## 5. 交互设计

### 5.1 关键流程

**回测流程**：
```
选择策略 → 配置参数 → 确认(含合规提示) → 执行(进度条) → 结果展示(含免责)
```

**订阅流程**：
```
选择方案 → 风险测评(未测评用户) → 确认订阅 → 支付 → 开通权限
```

**信号推送**：
```
选择策略 → 配置推送规则 → 确认(含合规提示) → 接收推送(每条含免责)
```

### 5.2 加载与反馈

- **骨架屏**：所有数据页面使用骨架屏加载
- **进度条**：回测等长时任务显示进度
- **空状态**：无数据时展示引导文案
- **错误状态**：友好的错误提示 + 重试按钮

## 6. 移动端适配

### 6.1 适配策略

- **桌面端优先**：核心功能在桌面端完整体验
- **移动端精简**：仅保留查看、监控、推送功能
- **关键操作**：回测创建、策略配置等复杂操作引导至桌面端

### 6.2 移动端页面

| 页面 | 功能 | 备注 |
|------|------|------|
| 仪表盘 | 市场概览 + 策略监控 | 简化版 |
| 信号推送 | 推送消息列表 | 核心功能 |
| 组合跟踪 | 持仓与收益 | 只读 |
| 账户设置 | 订阅管理 | 完整 |
