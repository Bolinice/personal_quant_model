## 📋 预处理顺序详解

### 正确顺序及原因

```
1. 缺失值处理
   ↓
2. 去极值
   ↓
3. 中性化 ⭐ 关键步骤
   ↓
4. 标准化
   ↓
5. 方向统一
```

### 为什么中性化必须在标准化之前？

**原因1：数学原理**
- 中性化使用回归取残差：`residual = factor - β * industry`
- 残差天然满足：均值 ≈ 0，但方差 ≠ 1
- 如果先标准化（均值=0，方差=1），再中性化，残差不再满足标准正态分布

**原因2：可比性**
- 标准化的目的是让不同因子具有可比性（均值=0，方差=1）
- 如果先标准化再中性化，中性化会破坏标准化的结果
- 正确顺序：先中性化（消除行业/市值影响），再标准化（统一量纲）

**原因3：实证验证**
```
错误顺序：标准差 = 0.9909（偏离1）
正确顺序：标准差 = 1.0000（符合预期）
```
