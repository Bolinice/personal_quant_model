# 申万行业历史调整记录模板

本CSV文件用于手动维护申万行业历史调整记录。

## 格式说明

```csv
ts_code,industry_code,industry_name,effective_date,expire_date
```

- `ts_code`: 股票代码（如 000001.SZ）
- `industry_code`: 行业代码（如 801780）
- `industry_name`: 行业名称（如 银行）
- `effective_date`: 生效日期（YYYY-MM-DD）
- `expire_date`: 失效日期（YYYY-MM-DD，留空表示当前有效）

## 使用方法

1. 填写历史调整记录
2. 运行同步脚本：
   ```bash
   python scripts/sync_industry_historical.py --csv data/industry_changes.csv --standard sw --level L1
   ```

## 注意事项

1. **历史调整记录**：申万行业分类会定期调整，需要维护完整的历史记录
2. **生效日期**：通常为每年年初或半年初
3. **失效日期**：当股票从某行业调出时设置，否则留空
4. **数据完整性**：确保每只股票在任意时点都有唯一的行业分类
