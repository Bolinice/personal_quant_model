## 路由结构

```
/app/strategies
├── /                          → StrategyList (策略列表)
├── /templates                 → TemplateStrategies (模板策略) ⭐ 新增
├── /new                       → StrategyForm (新建策略)
├── /:id                       → StrategyDetail (策略详情)
└── /:id/edit                  → StrategyForm (编辑策略)
```
