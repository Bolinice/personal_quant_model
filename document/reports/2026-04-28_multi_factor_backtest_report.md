## 一、策略概述

本策略采用 **自适应ICIR加权多因子选股 + Regime-aware权重调整 + ML增强融合 + 多周期择时** 框架，在三个主流宽基指数成分股池中分别运行增强策略。

### 优化演进

| 版本 | 核心优化 | 关键改善 |
|------|----------|----------|
| V1 | 基础ICIR加权 + 日线择时 | 基线 |
| V2 | +行业中性化 +换手率控制 +行业约束 +多周期择时 +回撤保护 | 回撤-5pp, 换手率-60% |
| **V3** | +分析师预期因子 +ML融合 +自适应ICIR +Regime-aware +成本优化 +行业偏离度约束 | Alpha增强+风控精细化 |

### V3优化项详解

| 优化项 | 说明 | 参数 |
|--------|------|------|
| 分析师预期因子 | consensus_revision/analyst_coverage/eps_surprise | 3个新因子 |
| ML增强融合 | LightGBM Walk-Forward + ICIR混合 | 0.6*ICIR + 0.4*ML |
| 自适应因子权重 | 滚动12期ICIR替代全样本ICIR | 每次调仓重算 |
| Regime-aware选股 | 趋势/震荡/防御三种市场状态切换因子权重 | 实时检测 |
| 成本优化 | 换仓成本过滤，预期收益超成本+0.5%才换仓 | 阈值0.5% |
| 行业偏离度约束 | 相对基准行业偏离≤5% | 使用指数行业权重 |

## 七、策略优化建议（下一期）

### 7.1 高优先级

1. **中证500 Alpha突破**：当前超额收益仍为负(-20.85%)，需引入更强的Alpha源
   - 供应链因子（上下游景气度传导）
   - 另类数据因子（卫星/舆情/专利）
   - 高频特征因子（微观结构因子）

2. **ML模型升级**：当前LightGBM Walk-Forward提升有限
   - 尝试Transformer时序模型
   - 因子交互特征自动生成
   - 集成多模型（XGBoost + LightGBM + NN）

3. **动态持仓数量**：根据市场波动率动态调整持仓数量
   - 高波动期增加持仓数降低集中度风险
   - 低波动期减少持仓数提升Alpha集中度

### 7.2 中优先级

4. **多空组合**：引入对冲机制，构建市场中性组合
5. **因子衰减监控**：实时监控因子IC漂移，自动剔除衰减因子
6. **自适应调仓频率**：根据信号强度动态调整调仓周期
